
2. **Access the Application**:

   - Admin Panel: http://localhost:8000/admin

3. **Test the Form**:
   - Go to http://localhost:5173

   - Submit your data

## Troubleshooting

If npm install is taking too long:
- Press Ctrl+C to cancel
- Try: `npm install --legacy-peer-deps`
- Or: `npm install --force`
